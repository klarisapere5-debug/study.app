# Study Planner Pro

🚀 Built with modern web technologies and deployed as a production-ready application.

A full-stack task management web application that allows users to securely manage tasks using authentication and a real-time cloud database.

## Live Demo
https://klarisapere5-debug.github.io/study.app/

## Repository
https://github.com/klarisapere5-debug/study.app

## Features
- User authentication (Firebase)
- Add, filter, and manage tasks
- Priority system (High / Medium / Low)
- Dark mode
- Real-time data storage (Firestore)

## Tech Stack
- TypeScript
- Vite
- Firebase Authentication
- Firestore Database
- GitHub Pages

- Building full-stack apps with Firebase
